# ગ્રાહક પાસેથી પૂછવાની માહિતી — Exact Price Estimate માટે

> **હેતુ:** Aluminium & Fabrication કામનો સાચો (accurate) **Estimate / Quotation** આપવા માટે ગ્રાહક પાસેથી નીચેની બધી details લેવી જરૂરી છે. જેટલી details ચોક્કસ, એટલો price estimate સાચો.

---

## 1. Basic / સામાન્ય માહિતી

- ગ્રાહકનું નામ, Mobile No., WhatsApp No.
- Site / કામની જગ્યાનું પૂરું **Address** (City, Area, Pin code)
- કામનો પ્રકાર — **Residential / Commercial / Industrial / Shop**
- કામ ક્યારે શરૂ કરવાનું છે — **Timeline / Urgency**
- **Budget** આશરે કેટલું છે (range)

---

## 2. કામનો પ્રકાર (Type of Work)

ગ્રાહકને પૂછો કે કયું કામ કરાવવું છે:

- [ ] **Window** (Sliding / Openable / Fix / Casement)
- [ ] **Door** (Sliding / Openable / Slide-Fold)
- [ ] **Partition** (Office / Glass partition)
- [ ] **ACP Cladding** (Front elevation)
- [ ] **Structural Glazing** / Curtain Wall
- [ ] **Railing** (SS / Glass / Aluminium)
- [ ] **Mosquito Net / Mesh** (Jali)
- [ ] **Toughened Glass** કામ
- [ ] **Skylight / Roofing**
- [ ] **Gate / Shutter**
- [ ] બીજું (Custom) — details લખો

---

## 3. Measurement / માપ (સૌથી અગત્યનું)

> **Note:** માપ વગર exact price આપવો શક્ય નથી. Site visit શ્રેષ્ઠ છે.

- દરેક Window/Door નું **Width × Height** (feet અથવા mm માં)
- કુલ કેટલા **Nos.** (quantity) જોઈએ છે — દરેક size ના કેટલા
- કુલ **Sq.ft / Sq.mtr** area
- **Floor level** — Ground / 1st / 2nd... (ઊંચાઈ scaffolding ખર્ચ વધારે)
- માપ ગ્રાહકે આપ્યું છે કે **Site visit** કરીને લેવાનું છે?

---

## 4. Aluminium Profile / Section Details

- **Brand / Company** — Jindal / Hindalco / Local
- **Series** — દા.ત. 2 Track / 3 Track, 27mm, 40mm વગેરે
- **Section weight** (kg/mtr) — Light / Medium / Heavy
- **Finish / Color:**
  - [ ] Powder Coated (કયો color — White / Brown / Black / Ivory)
  - [ ] Anodized (Silver / Champagne / Black)
  - [ ] Wooden Finish (PVDF / Sublimation)
  - [ ] Mill Finish (plain)

---

## 5. Glass Details (કાચ)

- **Type:**
  - [ ] Plain / Float Glass
  - [ ] **Toughened** (Tempered)
  - [ ] Frosted
  - [ ] Tinted (Blue / Green / Grey / Black)
  - [ ] Reflective
  - [ ] **DGU** (Double Glazed Unit)
  - [ ] Laminated
- **Thickness** — 4mm / 5mm / 6mm / 8mm / 10mm / 12mm
- કાચ ઉપર **Film / Sticker** જોઈએ છે? (Sun control / Privacy)

---

## 6. Mesh / Jali (Mosquito Net)

- જરૂરી છે? — હા / ના
- **Type:** SS Mesh / Fiber Mesh / Pleated (Openable) / Roller
- કયા window/door પર જોઈએ છે

---

## 7. Hardware & Fittings

- **Lock System** — Normal / Multipoint / Smart lock
- **Handle** type — Normal / Premium
- **Rollers** — Normal / Heavy duty (SS bearing)
- **Hinges / Friction stay** quality
- **Brand preference** — હા હોય તો કયો

---

## 8. ACP / Cladding (જો લાગુ પડે)

- **Brand** — Alstrong / Aludecor / Eurobond / Local
- **Thickness** — 3mm / 4mm
- **Grade** — FR (Fire Retardant) જોઈએ છે?
- **Color / Finish** — Solid / Metallic / Wooden / Mirror
- કુલ **Sq.ft** area
- Framing structure — GI / Aluminium

---

## 9. Installation / Site Conditions

- **Old material કાઢવાનું** (dismantling) છે? — હા / ના
- **Civil work / Repair** જરૂરી છે? (wall cutting, plaster)
- **Scaffolding** જોઈએ — height કેટલી
- Site સુધી material લાવવા **Transport / Lift** સગવડ છે?
- **Electricity & Water** site પર available છે?
- કામના કલાકો — Day / Night કોઈ restriction છે?

---

## 10. Commercial / Quotation Terms

- **GST** સાથે કે વગર quotation જોઈએ
- **Payment terms** — Advance %, Balance ક્યારે
- **Warranty / Guarantee** કેટલા વર્ષની જોઈએ
- **Material + Labour** બંને કે ફક્ત Labour (job work)
- **Drawing / Design** approval જોઈએ છે?
